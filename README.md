# Simon Says Game 🎮

A fun and interactive Simon Says memory game built using HTML, CSS, and JavaScript.

## Features
- Random color sequence generation
- Increasing difficulty with each level
- Score/level tracking
- Interactive animations and sound effects
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Play
1. Press any key to start the game.
2. Watch the sequence of colors displayed.
3. Repeat the sequence by clicking the buttons in the correct order.
4. The sequence gets longer each round.
5. The game ends when you click the wrong button.
